### JavaScript Access Patterns

| Data Structure | Access Syntax | Example |
|---|---|---|
| Object | `obj.key` or `obj['key']` | `user.name`, `user['age']` |
| Map | `map.get(key)` | `m.get('name')` |
| Array | `arr[index]` | `arr[0]` |
| Set | `set.has(value)` | `s.has(42)` |
