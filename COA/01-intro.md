**Processor**-> Active block of computer thatesponsible for following instructions making up a computer program. It performs basic arithmetic, logic, control, and input/output operations specified by the instructions.
**Memory**-> Component of a computer that stores data and instructions temporarily or permanently for use by the processor.

buses are digital connection between functional blocks. they can be serial or parallel.

harvard architecture -> separate memory and buses for instructions and data
von neumann architecture -> single memory and bus for both instructions and data
